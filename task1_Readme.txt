
Задание выполнено путем написания и исполнения скрипта на языке T-SQL.
Скрипт task1.sql прилагается.

--- Пояснения по заданию 1.2:

Для начала необходимо выполнить настройку:
- Получить доступ к дополнительным параметрам конфигурации (Show Advanced Options).
- Разрешить нерегламентированные распределенные запросы (Ad Hoc Distributed Queries), которые использует оператор OPENROWSET для доступа к удаленным источникам данных OLE DB.
- Установить свойства поставщика MS Office ACE OLE DB (AllowInProcess, DynamicParameters) для возможности работы с MS Excel.

После подготовки можно получить данные из таблицы Excel как со связанного сервера, а затем с помощью оператора MERGE синхронизаровать таблицу БД и таблицу Excel путем обновления уже существующих записей в dbo.Users и вставки тех записей, которые отсутствуют, предварительно сгенерировав уникальный идентификатор.
